# Sales
Built an Excel Sales Dashboard using the Power Pivot and Data Model

Sales Dashboard using the Power Pivot and Data Model, enabling efficient analysis across multiple tables and create relationships, measures, and calculations, enabling deeper insights into sales performance

 The dashboard delivers clear KPIs and visual insights into sales trends by many points to analyze.

Overall, this dashboard provides managers with a quick and accurate overview of sales performance, helping them make informed strategic decisions.
